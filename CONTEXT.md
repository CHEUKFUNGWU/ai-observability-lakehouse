# Domain Glossary

## Access Audit Event

A record of one user's attempt to act on a classified AI data resource, whether the attempt was granted or denied.

## Data Retention Event

A record that a retention policy archived, anonymized, or deleted one table partition. It describes policy enforcement, not the policy definition itself.

## Agent Handoff

The transfer of work or control from one agent run to a distinct child agent run. A handoff is not an internal step performed by the same agent.

## Platform Health Metric

A threshold-bearing observation about the operational state of a lakehouse component. It is distinct from business workload and AI runtime metrics.
